# Alien-Invasion
Cross-platform game that lets you lead the spaceship and kill aliens
## Getting Started

### Prerequisites
Ensure you've got python 3 and pip installed.

Use pip (package manager) to install all the requirements from requirements files:

From the console:
```
pip install -r windows-requirements.txt
```

## Running

Run run.py file

For windows:
```
python run.py
```
For linux and macos operating systems:
```
python3 run.py
```
